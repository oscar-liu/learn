Promise Api 方法

示例代码： 
        let p1 = new Promise((resolve, reject) => {
            resolve('p1 ok')
        })
        let p2 = Promise.resolve('p2 success')
        let p3 = Promise.resolve('p3 success')

all方法

            // all调用
            const allResult = Promise.all([p1, p2, p3])
            console.log(allResult)

这段代码会返回一个promise对象有三个成功数据

race方法


        // race调用， 谁先改变状态，就返回谁
        const result = Promise.race([p1, p2, p3])
        console.log( result) // 返回了p1 ok

谁先返回了状态就返回谁，抢占模式
